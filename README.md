# Higher Education Marketing Analytics Case Study

## Project Overview

This project analyses marketing performance across six postgraduate academic programs to identify conversion drivers, enrollment economics, and optimisation opportunities.

## Business Objective

The objective was to:

- Evaluate program-level marketing performance
- Identify scalable programs
- Improve conversion efficiency
- Optimise enrollment economics
- Generate actionable business recommendations

## Dataset

- 180,753 Leads
- 8,190 Admissions
- ₹12.04 Cr Marketing Spend
- 6 Academic Programs
- Analysis Period: 2024–2025

## Key Findings

### Portfolio Performance

- Admissions increased by 29% despite a 20% decline in lead volume.
- Conversion improved from 3.55% to 5.76%.
- Cost per enrollment reduced by 36%.

### Program Insights

- AI & ML delivered the strongest performance with the highest conversion rate and lowest enrollment cost.
- CSI demonstrated strong scale potential.
- Data Science & Engineering showed the weakest enrollment economics.
- Software Engineering required a strategic review due to high acquisition costs.
